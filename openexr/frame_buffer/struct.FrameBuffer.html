Insert a single channel into the FrameBuffer.

The channel will be given the name name.

data is the memory for the channel and should contain precisely width * height elements, where width and height are the dimensions of the FrameBuffer.

fn insert_channels<T: PixelStruct>(
    &mut self,
    names: &[&str],
    data: &'a [T]
) -> &mut Self

Insert multiple channels from a slice of structs or tuples.

The number of channels to be inserted is determined by the implementation of the PixelStruct trait on T. names should contain the names of each of those channels.

data is the memory for the channel and should contain precisely width * height elements, where width and height are the dimensions of the FrameBuffer.
